There are two protocols supported to access the different service operations: SOAP over HTTP or RESTful

SOAP based service

Web service with SOAP, please look to the following WSDL http://hostname:port/erx/serv/ErxDecisionServices.wsdl

RESTful services

The decision service can be exposed with RESTful service with JSON payload. Here are the operations offered

  • POST http://hostname:port/erxds/rest/erxtransaction/drugSelection: drug selection
  • POST http://hostname:port/erxds/rest/erxtransaction/drugQuantity: quantity computation
  • POST http://hostname:port/erxds/rest/erxtransaction/drugDaySupply: day supply
  • POST http://hostname:port/erxds/rest/erxtransaction/thirdParty: thirdParty
